*** ## WP_CLI\Utils ### format_items ```WP_CLI\Utils\format_items( $format, $items, $fields )``` Output items in a table, JSON, CSV, ids, or the total count ### write_csv ```WP_CLI\Utils\write_csv( $fd, $rows, $headers = array() )``` Write data as CSV to a given file. ### get_named_sem_ver ```WP_CLI\Utils\get_named_sem_ver( $new_version, $original_version )``` Compare two version strings to get the named semantic version. ### get_flag_value ```WP_CLI\Utils\get_flag_value( $args, $flag, $default = array() )``` Return the flag value or, if it's not set, the $default value. ### get_temp_dir ```WP_CLI\Utils\get_temp_dir()``` Get the temp directory, and let the user know if it isn't writable.
